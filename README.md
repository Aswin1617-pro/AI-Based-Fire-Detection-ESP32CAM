# 🔥 AI-Based Fire Detection System using ESP32-CAM

An AI-powered fire detection system that uses the ESP32-CAM for live video streaming and a YOLOv8 deep learning model for real-time fire detection. The system detects fire from live camera frames and displays the detection with bounding boxes and confidence scores.

## 📌 Features

- 🔥 Real-time Fire Detection
- 📷 Live Video Streaming using ESP32-CAM
- 🤖 YOLOv8 Deep Learning Model
- 🖼️ Bounding Box Detection
- 📊 Confidence Score Display
- ⚡ Fast and Accurate Detection
- 💻 Python + OpenCV Implementation

## 🛠️ Hardware Used

- ESP32-CAM
- FTDI Programmer
- USB Cable
- Jumper Wires

## 💻 Software Used

- Arduino IDE
- Python
- OpenCV
- Ultralytics YOLOv8
- VS Code
